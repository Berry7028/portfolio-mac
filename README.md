# プロジェクト概要

このプロジェクトは、**複数のポートフォリオを表示するWebサイト**です。  
UIはmacOS風にデザインし、Safari風のウィンドウで過去作成したポートフォリオを確認・閲覧できるようにします。

## 主なアイディア

- Safari風ウィンドウ内で各ポートフォリオサイトを閲覧可能
- メッセージアプリのようなUIから簡単にコンタクトを取れる仕組みも紹介
  - ※本物のメッセージアプリのように実装する案を検討中
- DB連携＆ログイン機能を追加し、採用担当者などと1対1でメッセージができる仕様にすれば技術アピールにもなる

## 開発スタンス

- コードは**自力で書く**ことを重視し、AIによる全自動生成は避ける
- AIにコード例やヒントを出してもらい、実装は自分で行う
- コードの可読性＆クリーンさを重視
- 補完（Tab補完等）のような支援ツールは積極的に活用  
  （今後のエンジニアの標準的な作業スタイルも意識）
